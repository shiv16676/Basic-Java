Postgresql:
************************
1.Open CMD
2. Go to C:\Alfresco\postgresql\bin
3. Enter the required command:
pg_ctl -D C:\Alfresco\alf_data\postgresql status
pg_ctl -D C:\Alfresco\alf_data\postgresql start
pg_ctl -D C:\Alfresco\alf_data\postgresql stop

pg_ctl -D k:\Alfresco\alf_data\postgresql start
